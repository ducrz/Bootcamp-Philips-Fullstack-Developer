Sobre o HTML5: O HTML5 é uma evolução do HTML. O HTML5 traz uma série de novidades para a web. HTML5 é a última versão da linguagem para descrever a estrutura da Web. O HTML está no coração de toda página web e a sua evolução é fundamental para todos os que trabalham com Web, seja front-end ou back-end. Gere sites mais fáceis, rápidos e concisos. HTML5 é a evolução do HTML. Financiado pelo pessoal da W3C e de outras empresas/instituições, o sucessor do HTML 4.01 chega com novas tags, novas funções e recursos para web application que prometem revolucionar o desenvolvimento desses tipos de sites. Você já começou a usar o HTML5? Se não, veja quais são as principais funcionalidades e se impactam ou não no seu trabalho. Nesta edição, você conhece as novidades da HTML 5, a linguagem de marcação mais importante para desenvolvedores web, além de saber os motivos que fizeram ela evoluir tanto. Descubra, também, como implementar os novos recursos da especificação e quais são as diferenças com relação ao HTML 4." Com o aumento do uso de dispositivos móveis como smartphones e tablets, as necessidades dos usuários mudaram: os visitantes às páginas web passaram a fazer buscas e consultas rápidas, que exigem interfaces amigáveis e compatíveis com dispositivos móveis. A versão atual do HTML não comportava estas evoluções. E não precisa se preocupar, pois o HTML5 é retrocompatível. Em paralelo com o trabalho no HTML5, a W3C tem trabalhado para melhorar a compatibilidade da Web e surgiram muitos estudos científicos. A tecnologia do HTML5 está amadurecendo e várias implementações estão sendo adotadas pelas principais empresas, web Giants como Google, Microsoft e Facebook e os browsers mais populares já o suportam. O HTML5 tornou-se parte da World Wide Web Consortium Recommendation em 2014.
